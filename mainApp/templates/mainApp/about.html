The developed software product solves the problem of searching for a specific image in a personalized user database. The project is a web service where you can add a title, description and hashtags to each of the images. Search and sort by given properties. Among other things, "Saved" automatically completes the description with recognizable text and generates a signature describing the processes from the image. Also, hashtags are automatically added, meaning objects recognized in the image.

A personalized image database is created by importing files from the user's device, as well as a special function of the web service - importing images from the "Saved Photos" album on the VK social network.
